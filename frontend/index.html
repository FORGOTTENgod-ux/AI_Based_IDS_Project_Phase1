How It Works

Data Collection

2

Feature Extraction

3

AI Analysis

Our trained Random Forest model analyzes the extracted features to classify traffic patterns.

4

Threat Detection

5

Visualization

View detailed results with interactive charts, probability scores, and sample flow predictions.

📡 Live Monitoring Workflow

1

Interface Selection

Select your network interface (e.g., Wi-Fi, Ethernet) and optionally apply packet filters to focus on specific traffic.

2

Real-Time Capture

Start capturing network packets in real-time from the selected interface. Packets are captured and processed as they flow through the network.

3

Flow Aggregation

Packets are grouped into network flows based on source/destination IPs, ports, and protocols. Flow features are extracted in real-time.

4

Continuous AI Analysis

Each aggregated flow is analyzed by the ML model in real-time to detect threats and classify traffic patterns as they occur.

5

Live Threat Alerts

Immediate alerts are displayed for detected threats. View live packet table, traffic graphs, and real-time statistics as monitoring continues.

6

Interactive Dashboard

Monitor live traffic graphs, view packet details in real-time, and track threat patterns through an interactive dashboard interface.

Frequently Asked Questions

What file formats are supported for upload?

+

We support PCAP (.pcap), PCAPNG (.pcapng), and CSV (.csv) file formats. PCAP files are automatically processed to extract network flow features, while CSV files should contain pre-aggregated flow data.

How accurate is the threat detection?

+

Our Random Forest model has been trained on diverse network traffic datasets and achieves high accuracy in detecting DoS attacks, port scans, and SQL injection attempts. The model provides probability scores for each prediction to help assess confidence levels.

Can I monitor network traffic in real-time?

+

Yes! The Live Monitoring feature allows you to capture and analyze network packets in real-time. You can specify network interfaces and apply filters to focus on specific traffic patterns.

What types of attacks can the system detect?

+

Currently, the system can detect three main attack types: Denial of Service (DoS) and Distributed DoS (DDoS) attacks, port and vulnerability scanning activities, and SQL injection attempts. We're continuously working to expand detection capabilities.

Is the system suitable for enterprise use?

+

Yes, the system is designed to handle enterprise-scale network traffic analysis. It can process large PCAP files and provides detailed reporting suitable for security teams and network administrators.

How does the AI model work?

+

The system uses a Random Forest machine learning algorithm trained on aggregated network flow data. It analyzes features like packet counts, byte totals, protocol information, and timing patterns to classify traffic as normal or malicious.
